# Workspace
---

Workspace projects for the following models and simulation methods:
- `bisse_r`          : BiSSE model (R and castor)
- `levy_r`           : Levy model (R and pulsR)
- `geosse_revbayes`  : GeoSSE model (RevBayes)
- `bisse_timehet_pj` : Time-heterogeneous BiSSE model (Phylojunction)
- `bdk_master`       : Birth-death model with density-dependent extinction (MASTER and masterpy)
- `sir_master`       : SIR model (MASTER and masterpy)
- `geosse_master`    : GeoSSE model (MASTER and masterpy)
